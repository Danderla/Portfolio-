Tech Stack

Here’s a breakdown of my tech stack—what I use, why I use it, and where each tool fits into my projects.

Frontend

  • HTML & CSS — clean, accessible UI basics.
  • JavaScript — interactivity and data fetching.
  • React (later) — components & state management.

Backend

  • Node/Express (later) — simple REST APIs.
  • Java Spring Boot (optional) — strong typing & structure.

Data

  • SQLite → PostgreSQL — start simple, scale to relational.

Dev Tools

  • Git & GitHub — version control and hosting.
  • VS Code — editor with helpful extensions.
